How to use

  1. Make sure your Ethereum account is ready.
  2. Select file using document selector.
  3. Hash of the file will be generated.
  4. Then, you can do:
    • `Sign` to sign the document and record it on the blockchain.
    • `Show Owner` to show document owner.
    • `Show Registrant` to show who first registers the document.
    • Put an Ethereum address and click `Check` to check whether a document has been signed by that address.
    • Put a target Ethereum address and click `Transfer` to transfer document ownership to that address.
