How to use

  1. Make sure the Ethereum account is ready.
  2. Select file using document selector.
  3. Hash and other information of the file will be fetched.
  4. Then, you can do:
    • `Sign` to sign the document and record it on the blockchain.
    • `Verify` to verify whether a party has signed the document or not.
    • `Transfer Ownership` to transfer the ownership of the document
